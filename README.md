# TrackMyCash

**TrackMyCash** - це вебзастосунок для обліку особистих фінансів.  
Система дозволяє додавати доходи та витрати, автоматично обчислювати баланс, працювати з категоріями, переглядати історію транзакцій, експортувати дані у PDF та аналізувати статистику за допомогою графіків.

Основний функціонал
- реєстрація та вхід користувача
- додавання доходів і витрат
- автоматичний підрахунок балансу
- перегляд усіх транзакцій
- редагування та видалення транзакцій
- робота з категоріями
- аналітика за тиждень, місяць і рік
- експорт транзакцій у PDF

Використані технології
- **C#**
- **ASP.NET Core**
- **Entity Framework Core**
- **SQLite**
- **HTML / CSS / JavaScript**
- **Razor Views**

Архітектура проєкту
Проєкт побудований за принципом розділення відповідальностей:
**Controllers** - обробка запитів користувача
**Services** - бізнес-логіка системи
**Models** - структура даних
**Data** - робота з базою даних через `ApplicationDbContext`
**Views** - інтерфейс користувача
**ViewModels** - передача даних між формами та контролерами


Як працює система
Користувач входить у систему.
Додає дохід або витрату через форму.
Дані передаються в контролер.
Контролер викликає відповідний сервіс.
Сервіс обробляє дані та зберігає їх у базу.
Після цього автоматично оновлюється баланс.
Користувач може переглядати транзакції, категорії та аналітику.

База даних
У проєкті використовується SQLite.
Основні сутності:
User
Transaction
Category
Зв’язки:
один користувач може мати багато транзакцій
кожна транзакція належить певній категорії
